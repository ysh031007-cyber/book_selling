# 눈 내리는 시간의 굴뚝 홍보 사이트

React + Vite + Tailwind CSS 기반 랜딩페이지입니다.

## 실행 방법

```bash
npm install
npm run dev
```

## 배포 빌드

```bash
npm run build
```

## 포함 파일

- `src/App.jsx`: 사이트 메인 컴포넌트
- `public/cover.png`: 동화책 표지
- `public/book.pdf`: 동화책 PDF
- `public/images/aster-city.png`: 도시 배경 이미지
- `public/images/nia-rooftop.png`: 니아 이미지
- `public/images/glass-bell-time.png`: 유리 종/시간 이미지
- `public/images/winter-night.png`: 밤 배경 이미지
